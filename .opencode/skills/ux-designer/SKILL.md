# UX Designer Skill

This skill provides specialized instructions for creating premium user interfaces using Vue 3, Tailwind CSS, and modern design principles.

## When to Use

- Creating new Vue components
- Designing landing pages or portfolio sections
- Improving existing UI/UX
- Building responsive layouts
- Implementing animations and transitions
- Any frontend design task

## Design System Reference

### Spacing Scale (8px base)
```
4, 8, 12, 16, 24, 32, 48, 64, 96, 128
```

### Typography Scale
- Display: 64-96px, weight 700
- H1: 48px, weight 700
- H2: 32px, weight 600
- H3: 24px, weight 600
- Body: 16px, weight 400
- Caption: 12-14px, weight 400

### Color Philosophy
- Maximum 2 accent colors
- Use subtle borders instead of shadows
- Prefer #000000 and #FFFFFF as base
- Background: #FAFAFA or #F5F5F5
- Text: #1A1A1A (primary), #6B6B6B (secondary)

### Radius System
Choose ONE and stick to it:
- Sharp: 0-2px
- Soft: 4-8px
- Rounded: 12-16px

## Premium Implementation Rules

1. **Never use glassmorphism** - Use subtle borders or minimal shadows
2. **Mobile-first** - Design for 375px first, then scale up
3. **Meaningful motion** - Transitions: 150-300ms, ease-out
4. **Accessibility** - WCAG AA contrast, focus states, semantic HTML
5. **No placeholders** - Use realistic content and images

## Component Checklist

Before finalizing any component:
- [ ] Accessibility verified (contrast, focus states)
- [ ] Mobile responsive (375px, 768px, 1024px)
- [ ] Semantic HTML structure
- [ ] Realistic content (no lorem ipsum)
- [ ] Premium visual quality
- [ ] Consistent spacing
