CÓMO EXPORTAR EL CV A PDF (DESDE HTML)
================================================================================

OPCIÓN 1 - Google Chrome (Recomendada):
1. Abre cv-yeison-alfonso.html en Google Chrome
2. Presiona Ctrl + P (Imprimir)
3. En Destino selecciona: "Guardar como PDF"
4. En Más configuración:
   - Tamaño de papel: A4
   - Márgenes: Ninguno
   - Marca la casilla "Gráficos de fondo"
5. Haz clic en Guardar
6. Nombra el archivo: YeisonAlfonso_ProductDesigner.pdf

OPCIÓN 2 - Microsoft Edge:
1. Abre el archivo en Edge
2. Ctrl + P
3. Selecciona "Guardar como PDF"
4. Asegúrate de marcar "Gráficos de fondo"
5. Guarda como: YeisonAlfonso_ProductDesigner.pdf

OPCIÓN 3 - Firefox:
1. Abre el archivo en Firefox
2. Ctrl + P
3. Selecciona "Microsoft Print to PDF" o "Guardar como PDF"
4. Guarda como: YeisonAlfonso_ProductDesigner.pdf

NOTA IMPORTANTE:
- El archivo HTML está diseñado para verse perfecto en A4
- Si al imprimir se corta contenido, ajusta la escala en la ventana de impresión
- Verifica siempre el PDF antes de enviarlo

CÓMO USAR LA VERSIÓN ATS (TXT)
================================================================================

PARA WORD:
1. Abre Microsoft Word
2. Abre cv-yeison-alfonso-ats.txt
3. Selecciona todo el texto (Ctrl + A)
4. Copia (Ctrl + C)
5. Pega en un documento nuevo de Word (Ctrl + V)
6. Aplica formato:
   - Fuente: Calibri o Arial, tamaño 11
   - Nombre: tamaño 16, negrita
   - Títulos de sección: tamaño 13, negrita, mayúsculas
   - Espaciado entre líneas: 1.15
   - Espaciado entre párrafos: 6pt después
7. Guarda como: YeisonAlfonso_ProductDesigner.docx

PARA GOOGLE DOCS:
1. Abre Google Docs
2. Pega el contenido del archivo .txt
3. Aplica formato similar al de Word
4. Descarga como .docx o .pdf

PARA FORMULARIOS DE APLICACIÓN:
1. Copia secciones individuales del archivo .txt
2. Pega en los campos correspondientes del formulario
3. Usa el resumen profesional para el campo "About me" o "Summary"
